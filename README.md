# homebrew-sinew

Homebrew tap for the [Sinew](https://github.com/sinew-mocap) mocap clusters.

```sh
brew tap sinew-mocap/sinew
brew install sinew-tui      # the slim driver (rebocap -> /sinew OSC + TUI)
```

Formulae build from source; pre-built Windows binaries live in each cluster's
GitHub release (see the Scoop bucket `sinew-mocap/scoop-bucket`).
